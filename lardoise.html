Instructions

1. Creer le Google Sheet

Creez un nouveau Google Sheet avec les colonnes suivantes en premiere ligne :

ColonneDescriptionExemple
DateDate du jour ou le plat sera affiche27/02/2026
CategorieType de platEntree, Plat, Dessert, Formule, Suggestion
NomNom du platVeloute de potiron
DescriptionDescription courte (optionnel)Creme fraiche, noisettes torrefifiees
PrixPrix en euros (chiffre seul)9
ActifOUI pour afficher, NON pour masquerOUI

2. Publier le Sheet

  1. Dans Google Sheets, allez dans Fichier → Partager → Publier sur le Web
  2. Selectionnez Document entier et le format CSV (.csv)
  3. Cliquez sur Publier

3. Partager le Sheet

  1. Cliquez sur le bouton Partager en haut a droite
  2. Dans "Acces general", selectionnez Tous les utilisateurs disposant du lien
  3. Le role doit etre Lecteur

4. Configurer l'ID

Copiez l'URL du Sheet depuis la barre d'adresse de votre navigateur et collez-la dans le champ "Configuration" ci-dessus. L'ID sera extrait automatiquement.

Formats de dates acceptes

  • JJ/MM/AAAA — ex : 27/02/2026
  • AAAA-MM-JJ — ex : 2026-02-27

Colonne Actif

Mettez OUI pour afficher un plat, NON pour le masquer temporairement sans le supprimer.
